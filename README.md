Welcome to **Auto-Dorker**, an automated Google Dorking and OSINT recon tool built for security researchers, ethical hackers, and digital hustlers. If you want to find exposed files, vulnerable servers, and hidden directories with **Zero-Mehnat**, this is your system hack.

## 🚀 Features (सिस्टम की ताकत)
* **100% Automated:** Type your target, and the tool fires the best dorks.
* **Termux Friendly:** Designed specifically to run smoothly on Android via Termux.
* **Lightweight & Fast:** No heavy dependencies. Pure Python power.
* **Zero-Mehnat Recon:** Saves hours of manual Google searching.

## 🛠️ Installation (Termux / Linux)
Apne Termux ya Linux terminal mein ye commands ek-ek karke paste karo:

```bash
# Update system
apt update && apt upgrade -y

# Install Python and Git
pkg install python git -y

# Clone the repository
git clone [https://github.com/AkashJii/Auto_dorker.py.git](https://github.com/AkashJii/Auto_dorker.py.git)

# Enter the directory
cd Auto_dorker.py
Tool ko start karne ke liye bas ye command dalo:python Auto_dorker.py
